# 13. Exam Preparations (Preparación de Examen)

## Resumen del Capítulo
Este capítulo final cubre consejos prácticos para prepararse y aprobar el examen AZ-900 Microsoft Azure Fundamentals. Incluye formato del examen, estrategias de estudio, tips para el día del examen, y consejos para maximizar puntuación.

## Temas Principales

### 1. Acerca del Examen AZ-900

#### a) Información General
- **Nombre oficial**: Microsoft Azure Fundamentals
- **Código**: AZ-900
- **Duración**: 85 minutos
- **Preguntas**: ~40-50 preguntas
- **Escala de puntos**: 0-1000
- **Puntuación Aprobatoria**: 700 puntos (70%)

#### b) Formato de Preguntas

**Tipos:**
1. **Multiple Choice** - Elige 1 de 4 opciones
2. **Multiple Select** - Elige múltiples opciones correctas
3. **Case Study** - Lee scenario, responde preguntas relacionadas
4. **Drag & Drop** - Arrastra items a categorías correctas
5. **Hot Area** - Click en imagen o diagrama correcta

#### c) Estructura del Examen

**Domains (Áreas):**
1. Cloud Concepts (~15-20%)
2. Azure Architecture (~15-20%)
3. Azure Services (~40-45%)
   - Compute
   - Networking
   - Storage
   - Database
   - Analytics
4. Azure Management (~15-20%)
   - Management tools
   - Governance
   - Compliance

### 2. ¿Qué Probar el Examen?

#### a) Conceptos Fundamentales
- Cloud computing definition
- IaaS vs PaaS vs SaaS
- Public vs Private vs Hybrid cloud
- Scalability, reliability, availability

#### b) Azure Architecture
- Regions y Availability Zones
- Resource Groups
- Azure Resource Manager
- Subscriptions y Tenants

#### c) Azure Services (Principal Foco)
- Compute: VMs, Scale Sets, App Services, Functions, Containers
- Storage: Blob, Disk, Files, Archive
- Database: SQL, MySQL, PostgreSQL, Cosmos
- Networking: VNets, Load Balancers, VPN, CDN
- Security: NSGs, Firewall, Identity

#### d) Management
- Azure Portal, CLI, PowerShell
- Cost management
- Compliance y governance
- Support options

### 6. Estrategia de Estudio

#### a) Activo vs Pasivo

❌ **Pasivo (No recomendado)**:
- Solo watch videos
- Solo read Microsoft Docs
- No hands-on practice
- Result: Forget quickly

✅ **Activo (RECOMENDADO)**:
- Take notes mientras estudias
- Do hands-on labs in Azure
- Practice exams frecuentemente
- Teach others (explain yourself)

#### b) Técnica Spaced Repetition
- Study topic
- Review 1 dia después
- Review 1 semana después
- Review 1 mes antes exam
- Result: Long-term retention

#### c) Flash Cards / Anki
- Crea flash cards de conceptos
- Review regular basis
- Particular util para:
  - Service names
  - Pricing models
  - Acronyms

#### d) Teach Others
- Explica conceptos a friend/colleague
- Escribe blog post
- Join study group
- Best learning method (pedagogy research)

### 9. Día del Examen

#### a) Antes de Empezar
- Check time arrival (llega 15 mins early)
- Take restroom break
- Deep breathing (calm nerves)
- Avoid caffeine (jitteriness)

#### b) Durante Exam
```
✅ DO:
- Read question carefully (2× si needed)
- Skip hard questions (return later)
- Manage time (85 mins ÷ 40-50 = ~1.5-2 mins/pregunta)
- Flag questions para volver
- Trust your instincts
- Use process of elimination

❌ DON'T:
- Rush (mistakes increase)
- Change answers constantly (usually wrong)
- Spend 5+ mins en 1 question
- Panic if unsure
```

#### c) Estrategia de Respuesta
```
1. Read question + todas las opciones
2. Quick mental assessment
3. Si sabes → Selecciona + move on
4. Si dudas → Eliminate obvias, guess best → flag → continue
5. Al final → Return a flagged questions
```

#### d) Gestión de Tiempo
```
85 minutos total:
- Browse questions: 5 mins
- Answer questions: 60-70 mins
- Review flagged: 10-15 mins
- Buffer/panic reserve: 5 mins
```

### 11. Preguntas Tipo del Examen

#### a) Identificación de Servicio

`Pregunta: "Your company needs a globally distributed database with sub-millisecond latency. What should you use?"`

**Respuesta**: Cosmos DB
- Identificar clave: "global", "sub-millisecond"

#### b) Pricing Scenarios

`Pregunta: "You have low-predictable workload. What's cheapest option?"`

**Respuesta**: Spot pricing o Functions pay-as-use
- Identificar: Unpredictable = Spot

#### c) Security

`Pregunta: "You need to control inbound traffic to subnet. What service?"`

**Respuesta**: Network Security Group (NSG)
- Identificar: "inbound traffic", "subnet"

#### d) Hybrid Cloud

`Pregunta: "Connect on-premises to Azure securely".`

**Respuesta**: VPN Gateway / ExpressRoute
- Identificar: "on-premises", "secure"

### 12. Errores Comunes en Examen

❌ **Evita**:
1. Confundir IaaS con PaaS
2. Pensade Regional vs Zone Redundancy
3. Load Balancer vs Application Gateway (application layer difference)
4. On-premises AD vs Azure AD
5. Leyend preguntas rapidamente (miss details)
6. Overthinking (second-guess correct answer)
7. Correr tiempo (no terminar)
