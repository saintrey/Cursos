## 📖 Descripción de Capítulos

### 🟢 Capítulos Core (Esenciales para Examen)

**01. Introducción** `500 líneas`
- Herramientas de Azure (Portal, CLI, PowerShell, Cloud Shell)
- ARM Templates y Azure Advisor
- Primeros pasos con Azure

**02. Cloud Concepts** `600 líneas`
- Definición de cloud computing
- Modelos: IaaS, PaaS, SaaS, Serverless
- Conceptos: Escalabilidad, Disponibilidad, Confiabilidad
- Cloud Público, Privado e Híbrido

**03. Azure Architecture** `550 líneas`
- Regiones y Availability Zones
- Resource Groups y Azure Resource Manager
- Jerarquía de organización
- Paired Regions para disaster recovery

**04. Compute** `700 líneas`
- Virtual Machines, Scale Sets, App Services
- Containers (ACI, AKS)
- Azure Functions (Serverless)
- Azure Virtual Desktop
- Matriz comparativa de servicios

**05. Networking** `650 líneas`
- Virtual Networks (VNets) y subnets
- Load Balancer vs Application Gateway
- VPN Gateway para conexión híbrida
- ExpressRoute para conexión privada
- CDN para distribución global

**06. Storage** `800 líneas`
- Blob Storage, Disk Storage, File Storage
- Archive Storage para datos no frecuentes
- Opciones de redundancia (LRS, ZRS, GRS, GZRS)
- Herramientas de migración de datos

**08. Authentication and Authorization** `700 líneas`
- Diferencia: Autenticación vs Autorización
- Azure Active Directory (AAD)
- Multi-Factor Authentication (MFA)
- Zero Trust Security Model
- Control de acceso basado en roles (RBAC)

**10. Security** `750 líneas`
- Defense in Depth (7 capas)
- Network Security Groups (NSGs)
- Azure Firewall
- DDoS Protection
- Encriptación, Key Vault, Azure Policy

**11. Pricing** `650 líneas`
- Modelos: Pay-as-you-go, Reserved Instances, Spot
- Calculadora de costos
- Optimización y TCO
- Free tier y créditos
