# 01. Introducción

## Resumen del Capítulo
La introducción al curso AZ-900 cubre la estructura general del examen, las herramientas fundamentales para interactuar con Azure y cómo prepararse para la certificación Microsoft Azure Fundamentals.

## Temas Principales

### 1. Visión General del Curso
- **Objetivo**: Proporcionar los conocimientos necesarios para aprobar el examen AZ-900
- **Certificación**: Primer paso en la carrera profesional con Azure
- **Beneficios de la certificación**:
  - Credibilidad profesional en el mercado
  - Acceso a red de profesionales certificados
  - Posibilidad de aumentar salario
  - Mayor confianza en aplicación de conocimientos

### 2. Estructura del Curso
Los temas principales cubiertos son:
- Cloud Concepts (Conceptos de Nube)
- Azure Architecture (Arquitectura de Azure)
- Compute (Computación)
- Networking (Redes)
- Storage (Almacenamiento)
- Database (Base de Datos)
- Authentication and Authorization (Autenticación y Autorización)
- Solutions (Soluciones)
- Security (Seguridad)
- Pricing (Precios)
- Support (Soporte)

### 3. Herramientas de Azure

#### a) Azure Portal
- Forma más común de interactuar con Azure
- Interfaz web intuitiva
- Acceso a todos los recursos
- Personalización con dashboards
- Control de acceso granular
- Gestión de costos integrada

#### b) Azure CLI
- Interfaz de línea de comandos
- Comandos de texto estructura lógica
- Multiplataforma (Windows, Mac, Linux)
- Ideal para automatización
- Confiable para scripting

#### c) PowerShell
- Herramienta para scripts avanzados
- Usa cmdlets (pequeños programas)
- Integración con Azure Resource Manager
- Preferido por profesionales IT
- Lenguaje propio de programación

#### d) Azure Cloud Shell
- Shell interactivo accesible desde navegador
- Opción Bash (CLI) o PowerShell
- Accesible desde web, móvil o cualquier dispositivo
- Autenticado y 100% seguro
- Herramientas comunes incluidas
- Almacenamiento propio para persistencia

#### e) Azure Mobile Apps
- Disponible para iOS y Android
- Información rápida en movilidad
- Alertas y estado del sistema
- Acceso a Cloud Shell desde el teléfono
- Sincronizado con Azure Resource Manager

### 4. ARM Templates (Plantillas de Azure Resource Manager)
- **Definición**: Forma de describir qué recursos crear/gestionar en Azure
- **Formato**: JSON
- **Ventajas**:
  - Idempotentes (mismo resultado al aplicar múltiples veces)
  - Automatización y evita errores humanos
  - Control de versiones
  - Reutilizable
  - Declarativo (especificas qué quieres, no cómo)
  - Replicación y consistencia

### 5. Azure Advisor
- Servicio gratuito que viene con Azure
- Proporciona recomendaciones para:
  - Mejorar disponibilidad de recursos
  - Reducir costos
  - Aumentar confiabilidad
  - Mejorar seguridad
  - Excelencia operativa
- Analiza infraestructura automáticamente

### 6. Creación de Cuenta Azure Gratuita
- Require email y verificación de número de teléfono
- Necesita tarjeta de crédito (para verificación, NO cobra sin autorización)
- Incluye crédito inicial gratuito
- Acceso a miles de imágenes y configuraciones

## Puntos Clave para el Examen

✅ **Recuerda**:
- Azure Portal es el punto de entrada más común
- CLI es ideal para automatización
- PowerShell para scripts complejos
- Cloud Shell combina lo mejor de ambos
- ARM Templates para infraestructura como código
- Azure Advisor proporciona mejoras automáticas
- La certificación AZ-900 es el primer paso
